# Employee-Reports

## A simple application using REST APIs, which will be a reporting system of a company and will be able to handle simple information storage and retrieval requests for employeesâ€™ reports. It is written in Java 11 with Spring Boot and it holds all records in memory database (H2 database).

### To run the server please navigate to project solution's folder and run via cmd the following command: 'mvn spring-boot:run'.   Server will start using embedded Tomcat on your localhost:8080. 
### Important! First run 'mvn clean install' to build the project and run the tests.
